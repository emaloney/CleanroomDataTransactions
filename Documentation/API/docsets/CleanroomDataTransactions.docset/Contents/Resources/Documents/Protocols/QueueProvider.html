QueueProvider

public protocol QueueProvider

Certain DataTransactions perform asynchronous operations using Grand Central Dispatch. Such DataTransactions will typically be given a QueueProvider at instantiation to supply a dispatch_queue_t when needed.

By supplying a QueueProvider rather than a specific queue, reusable DataTransactions can be supplied with different queues over time.
