URLTransaction

open class URLTransaction: DataTransaction

A DataTransaction that uses an URLRequest to request data from (and potentially send data to) a service at a given URL.

A successful transaction produces an Data instance, and if the request was sent via HTTP or HTTPS, the transaction metadata will contain an HTTPResponseMetadata instance.
