ApiDocTransaction

public class ApiDocTransaction<T>: JSONTransaction<T>

Uses a wrapped URLTransaction to connect to an apidoc.me-style RESTful JSON network service.

Because the root object of a JSON document may be one of several types, a successful ApiDocTransaction produces the generic JSONDataType. The PayloadProcessingFunction function is used to produce the expected type.
