FetchImageTransaction

public class FetchImageTransaction: GenericDataProcessingTransaction<PlatformImageType>

Attempts to construct a platform-appropriate image object by fetching image data from the given URL.

Note

PlatformImageType represents the image type appropriate for the runtime platform. On iOS and tvOS, it maps to UIImage; on Mac OS X, it’s NSImage.
